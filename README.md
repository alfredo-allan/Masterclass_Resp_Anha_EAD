📱 Página Web Educativa - CSS Masterclass

Projeto desenvolvido como parte do material didático para ensino de CSS, abordando desde fundamentos até técnicas avançadas de responsividade e adaptabilidade web.
✨ Sobre o Projeto

Página web moderna e interativa que transforma conceitos técnicos de CSS em uma experiência de aprendizado visual e dinâmica. O conteúdo aborda:

    📚 Evolução do CSS (CSS1, CSS2, CSS3)

    🎯 Conceitos-chave com analogias do mundo real

    💡 Casos de uso e exemplos práticos

    ⚡ Flexbox vs Grid Layout

    📱 Estratégia Mobile-First

    🔧 Pré-processadores (SASS/LESS)

🛠️ Tecnologias Utilizadas

    Bootstrap 5

    FontAwesome 6

    Anime.js

    HTML5 & CSS3

🚀 Como Visualizar

    Clone o repositório

    Abra o arquivo index.html no navegador
